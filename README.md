# Landing Page

Bu proje, **TailwindCSS** kullanılarak tasarlanmış modern bir tanıtım (landing) sayfasıdır.  
Basit ve sade bir kullanıcı arayüzü örneği sunar.  
Proje herhangi bir framework gerektirmez — yalnızca HTML ve CDN üzerinden Tailwind kullanır.
